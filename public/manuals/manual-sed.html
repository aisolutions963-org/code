Section 1

Introduction

Welcome to WoodWings PMS. This manual covers everything a SED user needs to know to operate the system effectively.

What is WoodWings PMS?

WoodWings PMS is the internal project management system for the WoodWings business. It tracks every client project from the first call through to handover and maintenance. Tasks, approvals, documents, payments, and team coordination all live in this single system.

Your Role: SED (Sales & Engineering Department)

As a SED user, you are the primary point of contact between clients and the rest of the company. Your responsibilities in the system are:

  • Recording client intake (F1 form) on first contact
  • Managing quotations, samples, design proposals, and site visits
  • Following up with clients across the project lifecycle
  • Submitting F5 quotation details once a project is confirmed
  • Ordering materials via F3 form when needed
  • Conducting QC checks at the installation site
  • Signing off on handover alongside the client

How to Log In

1
Open the systemNavigate to the application URL in your browser.
2
Enter your credentialsType your email address and password on the login page (/login).
3
You are redirectedAfter successful login, the system sends you directly to /dashboard/sed — your SED dashboard.

How to Log Out

Click your name or the menu at the bottom of the left sidebar, then select Log Out. You will be returned to the login page.

Session SecurityYour session stays active for a set period. If you close the browser and return, you may need to log in again. Never share your password.

Section 3

Task Management

Tasks are the core unit of work in WoodWings PMS. Understanding how they flow is essential to keeping projects moving.

Task Status Flow

Locked → To Do → In Progress → Pending Approval → Completed
StatusMeaningWhat you should do
LockedTask exists but a prerequisite task must complete first.Nothing — it will unlock automatically.
To DoReady to start. All prerequisites met.Open the task and set it to In Progress when you begin.
In ProgressYou are actively working on it.Complete the required action (form, call, etc.) then mark complete or submit for approval.
Pending ApprovalYou submitted — waiting for manager sign-off.Monitor the Approvals tab. If rejected, you will be notified by email.
CompletedDone. Next tasks in the chain will unlock.Nothing — move to the next task.

How to Start a Task

1
Find it in My TasksAll To Do tasks appear in your task list.
2
Open the task cardClick the task title to expand its detail panel.
3
Set to In ProgressClick the status button and select In Progress.
4
Complete the workFill in required fields, attach documents, or make the client call as the task requires.
5
Mark complete or submitClick Complete or Submit for Approval depending on the task type.

Tasks That Require Manager Approval

Some tasks (marked with a shield icon) require a manager to approve before they count as complete. When you submit one of these:

  • The task moves to Pending Approval
  • The manager sees it in their approval queue
  • If Approved: the task completes and the next task unlocks
  • If Rejected: the task returns to To Do and you receive an email with the manager's comment
Do not leave tasks in "In Progress" indefinitelyIf a task stays In Progress with no action for 3 days, it triggers an escalation to the manager and may cause the project to be marked Not-Approved.

Follow Up Task — Outcome Fields

On any Follow Up task card, you will see extra fields to record the outcome of client contact:

  • Follow-Up Outcome — what the client said (dropdown)
  • SED Note — free text note about the conversation
  • Call Count — number of attempts made
Always record outcomesEven if the client did not answer, update the Follow-Up Outcome field. This creates an audit trail and helps the manager understand project health.
Section 5

Project Workflow — Your Slice

SED touches every phase of a project. Here is a step-by-step guide to your role across all stages.

Phase 1: Preparing

This entire phase is driven by SED. You move the project forward based on what the client needs.

Receive first client contact
↓
Create project → Fill F1 Form (client intake)
↓
Choose one or more paths:
Make Quotation
↓
Quotation Approved gate
Order / Select Sample
↓
Sample Approved gate
Concept Design / Proposal
↓
Concept Design Approved gate
↓ (all chosen gates pass)
"Call the Client" task unlocks → Superadmin action
↓
Outcome: Approved / Needs Review / Refused
↓ Approved
Phase 2 begins — F4 + F5 submitted
3-Day Inactivity RuleIf no task is moved to In Progress within 3 days of the last action, the project is automatically escalated to the manager and may be marked Not-Approved. Keep tasks moving.

Phase 2: Open (Per Item)

Once F5 is submitted, each quoted item gets its own task chain. For each item, you may need to:

  • Conduct a site visit to take measurements
  • Select or order a material sample
  • Submit or review a design for approval
  • Obtain client approval to start fabrication

Phase 3: Working

Your involvement during fabrication and installation:

  • You receive notification when Fabrication Done — you and the fixing team have 2 days to inspect items and tools
  • Add any inspection comments on the task card
  • Conduct the QC Check at Site after installation is complete

Phase 4: Closing

  • Final QC check at site — mark QC done and attach site photo
  • Notify client to pay the final payment
  • Participate in or witness the F6 / Handover signing
Section 6

Common Scenarios

Scenario 1 — A new client calls. What do I do?

1
Go to My Projects → Create New Project.
2
Fill in the F1 intake form with the client's name, phone, location, and project description.
3
Submit. The system creates the project and your task list populates immediately.
4
Open the First Call task, mark it In Progress, then Completed.
5
Choose the next action tasks: Make Quotation, Order Sample, etc. Start working them.

Scenario 2 — The client approved the quote. What's next?

1
The Call the Client task is handled by the Superadmin who selects "Approved".
2
You receive a notification. The project moves to Open stage.
3
Submit the F5 Quotation Items form on the project page with all item details and prices.
4
Per-item task chains now appear. Work through each item's Phase 2 actions.

Scenario 3 — My task was rejected by the manager

1
You receive an email notification with the manager's rejection comment.
2
The task returns to To Do in your task list.
3
Read the manager's comment, make the necessary correction, and resubmit.

Scenario 4 — I need to order materials for a project item

1
Open the project in My Projects.
2
Click F3 — Order Materials.
3
Fill in the purpose and add material rows (item name, supplier, quantity, unit, needed-by date).
4
Submit. The material order is created and visible to the fabrication team.

Scenario 5 — Fabrication is done. What do I do with the 2-day check?

When fabrication marks the job done, the system creates a task for you (and the installation team) to inspect the items and tools within 2 days. Open that task, review the items, add any comments or concerns in the task notes field, and mark it complete. If something is missing or damaged, note it clearly — the manager will see it before scheduling the delivery.

Section 7

Permissions Reference

What you CAN do

  • Create new projects (F1 form)
  • View and action all tasks assigned to you
  • Submit F5 quotation items on your projects
  • Submit F3 material orders
  • Schedule and document site visits
  • Record follow-up outcomes and notes
  • Mark QC checks complete and attach photos
  • View your own project list and task history

What you CANNOT do

  • Approve or reject tasks (manager action only)
  • Advance a project stage manually
  • Set the "Call the Client" outcome (Superadmin only)
  • View payment amounts or financial records
  • Create or manage user accounts
  • Access the reports / download section
  • View fabrication or installation task details
  • Access the superadmin dashboard

Section 8

Troubleshooting

ProblemLikely CauseSolution
A task I expect to see is not in my listIt is still Locked (waiting for a prerequisite), or it is assigned to a different roleCheck the project page — Locked tasks are listed there. If you believe it should be assigned to you, contact the manager.
I cannot change a task statusThe task may be manager-gated or LockedIf gated, submit it for approval. If Locked, wait for the prerequisite to complete.
The project is showing "Not-Approved"The 3-day inactivity rule triggered, or the client declined via the Call the Client taskContact the manager or superadmin to review. They can reassess and potentially re-open the project.
F5 form is not appearing on the projectThe project may not yet be in Open stage, or you may not be the assigned SEDConfirm the project stage. If it is still Preparing, the client approval (Call the Client) has not been set yet.
I submitted a task but it shows "Pending Approval" with no actionThe manager has not reviewed it yetNormal — wait. If urgent, contact the manager directly. The task will appear in their approval queue.
I attached the wrong document to a taskN/AOpen the task, remove the incorrect attachment, and upload the correct one before marking complete.
A follow-up task keeps resettingFollow-up tasks may be part of a recurring check loopComplete the task fully (including outcome field) each time. If it keeps re-appearing, contact the superadmin.
