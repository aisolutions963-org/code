SED Manual

Manage your projects, record client interactions, and track follow-ups from start to close.

Dashboard Overview

Your dashboard shows only your own projects and tasks.

  • Task counters at the top — your open, in-progress, and pending approval tasks.
  • Commission card — your estimated commission based on amounts collected on your projects (1.5%).
  • Recent projects — a quick-access list of your most recently active projects.

Home Page

The home page (house icon) shows announcements targeted at you and the system calendar. Check here daily for pinned messages from management.

My Tasks

The My Tasks view shows all tasks assigned to you across every project.

Reading a Task Card

1Each task card shows the project name, task name, department, and current status badge.
2Click a card to expand it.

The expanded view shows all editable fields for that task type. Common SED fields include:

StatusChange to update progress
Call CountHow many times you've called
Post-Visit OutcomeWhat happened after the visit
SED NoteYour private note for this task
Document LinkAttach a Google Drive or Dropbox URL
Follow-Up DateWhen to follow up next
3Update the status and any relevant fields, then save.

Some tasks require manager approval when you submit them — they change to Pending Approval and the manager is notified.

Admin Note

If the superadmin has left a note on a task, it appears as a pinned amber banner at the top of the expanded card. You can read it but not edit it.

Priority Flag

If a task has a red 🚩 flag, it's been marked urgent by the manager or superadmin. Address flagged tasks first.

Client Approvals

Sidebar → Client Approvals shows tasks where you need to record whether the client has approved something — a design, a sample, or a quotation.

Waiting for Client

Tasks where you're still waiting for the client's feedback. The list shows the project name, task name, and how many days it's been waiting.

Recording an Approval Outcome

1Find the task and expand it.
2Fill in the approval outcome field.

Options vary by task type — for design approvals: Approved / Needs Revision / Rejected. For quotations: Accepted / Negotiating / Rejected.

3Set the status to Completed and save.

The task moves to the Decisions Recorded section and may trigger the next task to unlock.

Some approvals trigger a superadmin decision — after you mark it complete, the superadmin must review it before the project advances. You'll see this reflected in the project's task list.

Site Visits

Sidebar → Site Visits shows all your projects that require an on-site visit.

Scheduling a Visit

1Find the project and expand the visit task.
2Set the Visit Date field to when you plan to go.

The address shows on the card. Click the map icon to open the location in Google Maps.

3After the visit, fill in the Post-Visit Outcome and update the status.

QC Checks

Sidebar → QC Checks shows quality control inspection tasks assigned to you. These typically happen after fabrication or installation is complete.

Expand a QC task to record the inspection outcome and any site observations. Mark the task Completed when the inspection passes, or In Progress with a note if issues are found.

My Projects

Sidebar → My Projects shows every project assigned to you, with stage, client info, and quick-action buttons.

Searching and Filtering

Use the search bar to filter by project name or client name. Use the stage filter tabs to narrow to a specific stage.

Manager Notes

If the manager has left a note on a project, a note icon appears on the project row. Click it to read the manager's message.

Action Buttons per Project

  • F5 — Add or view quotation line items for this project
  • F3 — Submit a material order request
  • View — Open the full project detail page

Project Detail Page

Click any project name to open its detail page. Two tabs are available.

Tasks & Items Tab

All project tasks in a board view. Expand any task to update it. Production items show the fabrication progress.

Project Report Tab

A summary of the project's financials. You can see total collected and remaining balance — but you cannot edit payment records. Only amounts are shown, not method details.

Materials View

Sidebar → Materials shows all pending material orders across your projects. Use this to check on the status of orders you've submitted (F3).

Each row shows the item name, project, required date, quantity, status, and supplier. Orders are read-only here — contact the manager if an order needs to be changed.

Client Requests

Client Requests (sidebar → Client Requests) is where you create and track follow-on work for your clients. Three request types:

TypeWhen to use
TradeClient wants additional items beyond the original project scope
MaintenanceRepair or warranty work on a project that's already closed
VarianceScope change that needs to be tracked separately

Creating a Trade Request

1Click "+ New Request" and choose type: Trade.
2Select the parent project.

The client name auto-fills from the linked project.

3Describe the additional work needed and submit.

New tasks are generated for the request automatically.

Creating a Maintenance Request

1Click "+ New Request" and choose type: Maintenance.
2Select the project that needs warranty / repair work.

Only closed or warranty-stage projects appear in this list.

3Describe the issue and submit.

Tracking Request Progress

Each request row shows a task progress ring. The ring fills as tasks inside the request are completed. Click the row to see the full task list for that request.

Task Status Reference

Locked → To Do → In Progress → Pending Approval → Completed
StatusWhat it means
LockedA previous task must finish before this one opens
To DoReady to work on — nothing is blocking it
In ProgressYou're actively working on it
Pending ApprovalYou've submitted it — waiting for manager or superadmin to sign off
CompletedDone — may unlock the next task

FAQ

How do I record that a client approved the design?

Go to Client Approvals in the sidebar. Find the design approval task in the "Waiting for Client" list and expand it. Set the approval outcome to Approved, then change the status to Completed and save. The task moves to "Decisions Recorded" and may trigger the next step in the project.

What's the difference between Trade, Maintenance, and Variance requests?

Trade is for extra items the client wants added to an active project. Maintenance is for repair or warranty work after a project is closed. Variance is for scope changes that need to be tracked separately from the main contract. When in doubt about which to use, ask your manager.

How do I link a Trade request to a project?

When creating a Trade request, there's a "Parent Project" field. Select the original project from the dropdown. The client name auto-fills from the linked project so you don't have to enter it again.

A task is locked — when will it unlock?

A Locked task is waiting for another task to be completed first. Expand the locked task — it usually shows which task is the prerequisite. Check with the responsible team member on that blocking task. Once they complete it, your task will automatically unlock.

How do I leave a note for the manager on a task?

Expand the task card and find the SED Note field. Type your note and save — the manager and superadmin can see this note when they review the task. If the task requires approval, your note is visible in their approval queue.

Why did my task go to Pending Approval instead of Completed?

Some tasks require manager or superadmin sign-off before they can be marked done. When you set these tasks to Completed or submit them, they automatically switch to Pending Approval and the responsible person is notified. You'll see it move to Completed once they approve it.

How do I see what payments have been collected on my project?

Open the project from My Projects, then click the Project Report tab. You can see total collected and remaining balance. You cannot see payment method details or edit payment records — those are managed by the manager and superadmin.
