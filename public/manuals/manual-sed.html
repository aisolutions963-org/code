1. What Is This System?

WoodWings PMS is the company's project management system. It tracks every project from the moment a client enquiry arrives, through design, quotation, production, installation, and final handover.

As a member of the SED team (Sales, Estimating & Design), your role is to handle the client-facing side: create projects, manage quotations, follow up with clients, schedule site visits, and hand work over to production once the client says yes.

Everything is organised as tasks. Each step you need to take is a task card. When you finish one, the system automatically opens the next one in the chain.

2. How to Log In

  1. Open your web browser and navigate to the WoodWings app URL given to you by your manager.
  2. Enter your email address and password in the login fields.
  3. Click Sign In.
  4. You are taken directly to the SED Dashboard. No role selection needed.
📸

[SCREENSHOT: Login page — email field, password field, Sign In button]

Forgot your password? Ask Kanaan (the owner / superadmin) to reset it for you. There is no self-service reset link.

4. Understanding Tasks

What Is a Task?

A task is a single action card for a project. Examples: "Visit Site to Gather Details", "Make Quotation", "Call the Client". Each task has a name, a status, and sometimes a special form to fill in.

Task Statuses

StatusWhat It MeansAction Required?
LockedWaiting for another task to finish first. Hidden until it unlocks.No — wait
To DoReady for you. The task is sitting in your queue.Yes — start working
In ProgressYou have started this task.Keep going
Pending ApprovalYour part is done. Waiting for the manager to approve.No — wait for manager
CompletedFully done. No further action.No

Project Stages

Every project moves through stages as it progresses:

StageWhat It Means for SED
PreparingJust created. You are working on quotation, site visits, and samples.
OpenClient approved. Production is active. You submit F5 quotation items.
Installation CompletedInstallation done. Waiting for final payment and formal handover.
Closed & Valid MaintenanceProject closed. 1-year warranty period is running.
Not-ApprovedClient declined. No further tasks.

Parallel Tasks Are Normal

You will often see several tasks open at the same time for the same project — for example, "Make Quotation", "Order Sample", and "Site Visit" may all be open together. This is completely normal. They are different actions that happen in parallel. Work through each one independently.

The Amber Banner on a Task

If you open a task and see an amber (yellow/orange) banner at the top, it is a special note from Kanaan (the owner). Read it carefully — it contains specific instructions for that task. You cannot edit or remove it.

5. How to Work a Task

  1. Find the task in your Tasks tab. Tasks are grouped by project.
  2. Click the task card to open it. You will see the task name, any instructions, and fields to fill.
  3. Change the status to "In Progress" while you are working.
  4. Fill in the required fields. For a site visit, enter the visit date. For a follow-up call, enter the outcome.
  5. Mark as "Completed" when done. The system unlocks the next task automatically.
📸

[SCREENSHOT: Task card open — status dropdown, required fields, Complete button]

Tasks that need manager approval: Some tasks are gated — when you click Complete they move to Pending Approval instead of Completed. This is by design. The manager will review and either approve or reject it. You will receive a notification either way.

6. Special Task Types

Some tasks have built-in forms or buttons that replace the normal "fill a field and mark done" approach. Here are the ones you will encounter as a SED.

Make Quotation

When you see a task named "Make Quotation", you need to record the quotation details.

  1. Click the task card to open it.
  2. Fill in Quotation Number and Quotation Reference.
  3. Click Save & Complete. The task closes automatically.
📸

[SCREENSHOT: Make Quotation task panel — two input fields and Save & Complete button]

Order Sample

This task asks: does the team already have the material in stock?

  1. Click the task card to open it.
  2. You see two buttons:
    ✓ We Have It — Send to Fabrication — material is in stock.
    ✗ Need to Order — Request F3 material order — need to order it.
  3. Click the correct button. The system routes the work automatically based on your answer.
📸

[SCREENSHOT: Order Sample panel — two large action buttons (green / orange)]

Call the Client (Decision by Kanaan)

The "Call the Client" task is decided by Kanaan (superadmin), not by the SED team. However you will be notified of the outcome. There are three possible decisions:

DecisionWhat Happens
ApprovedClient agreed. Project advances to Open phase. New tasks are generated for all quotation items.
Needs ReviewClient wants changes. All action tasks reset to To Do so SED can rework and resubmit.
RejectedClient declined. Project is marked Not-Approved. No further tasks are created.
You will receive a notification when Kanaan records a decision on any Call the Client task.

Follow-Up Task

A follow-up task appears when you need to call the client and record what happened.

  1. Open the task and change status to In Progress.
  2. Make the call.
  3. Select the outcome from the Follow-Up Outcome dropdown.
  4. Mark the task as Completed.
⚠ Warning — "Reject Project" outcome: One of the dropdown choices is "Reject Project". If you select this and save, the project is immediately marked as Not-Approved. Only use this if the client has definitively declined. You cannot undo it yourself — you would need to ask Kanaan to reopen the project.

Site Visit Task

  1. Click the task card to open it.
  2. Set the Task Start Date to the planned visit date.
  3. Go to the site.
  4. Return to the task. Enter the Post-Visit Outcome — what you found and what was discussed.
  5. Attach any photos or documents using the document link fields.
  6. Mark the task as Completed.
Use the Site Visits tab to see all your upcoming visits with the Google Maps link for each location.

F5 — Add Quotation Items

Once a project moves to the Open phase, you must submit the formal quotation line items.

  1. Go to the Projects tab.
  2. Find the project and click F5 — Add Quotation Items.
  3. Fill in the form — one row per piece of furniture or scope item (name, quantity, unit price, notes).
  4. Click Submit. The system automatically creates per-item task chains.
📸

7. Notifications

The system notifies you when something needs your attention — a task approval, a new task assigned, a client decision recorded by Kanaan, or a weekly project review reminder.

Click the bell icon in the sidebar to open the Notifications page.

📸

[SCREENSHOT: Notifications page — list of unread (blue dot) and read notifications, "Mark all read" button top-right]

  • Unread notifications have a blue dot and a blue background.
  • Click any notification to mark it as read.
  • Mark all read button (top right) clears everything at once.
  • Use the All / Unread toggle to filter the list.
  • A weekly reminder is sent every Friday morning automatically — no action needed unless a specific project is flagged.

8. Frequently Asked Questions

Why can't I see a task I'm expecting?

Tasks that are Locked are not visible until a prerequisite task is completed. You don't need to do anything — the system unlocks them automatically. If you believe a task is missing entirely, check with your manager.

I clicked Complete but the task didn't close — why?

Either a required field is still blank, or the task uses a special form (like Make Quotation or Order Sample) that has its own submit button. You must use that button — the regular status dropdown won't complete these tasks.

I accidentally selected "Reject Project" in a follow-up — can I undo it?

No, you cannot undo it yourself. Contact Kanaan immediately and ask him to reopen the project from the admin dashboard. He can do this from the Superadmin Projects page using the Reopen (↩) button.

What is the amber/yellow banner I see on some tasks?

It is a personal note left by Kanaan (the owner) for that specific task. It usually contains a special client instruction, a reminder, or something to be careful about. Read it carefully and follow the instructions. You cannot edit or remove it.

Why are there multiple tasks open for the same project at once?

This is by design. In the Preparing phase, several tasks run in parallel — "Make Quotation", "Order Sample", and "Site Visit" can all be open at the same time because they are independent actions. Work through each one separately.

When does the F5 button appear on a project?

The F5 — Add Quotation Items button only appears once the project is in the Open stage (i.e., after Kanaan has recorded an "Approved" decision on the Call the Client task). If the project is still in Preparing, the button will not be visible yet.

I received a "Needs Review" outcome on Call the Client — what do I do?

All your action tasks (quotation, design, etc.) have been reset to To Do. Review the project, make the necessary changes the client requested, and resubmit your work. Kanaan will then make a new decision.

The page is very slow or showing an error — what do I do?

The app auto-refreshes every 30 seconds. If you see an error banner, click the Retry button that appears. If the problem persists for more than a few minutes, contact your administrator (Kanaan or IT support).
