Manager Manual

Oversee projects, approve tasks, manage payments, and keep operations moving.

Dashboard Overview

Your dashboard opens to a summary of active work and the items that need your immediate attention.

  • Open Tasks — total tasks across all projects that are in To Do or In Progress.
  • Awaiting Approval — tasks submitted by the team that need your sign-off.
  • Active Projects — projects currently in Preparing, Open, or Production stages.

Below the cards, a task list shows your pending approvals sorted by most recent. Below that, quick links to recent projects.

Home Page

The home page (house icon in the sidebar) shows announcements from the superadmin and a system calendar. Pinned announcements appear at the top with an amber highlight.

Pipeline

Sidebar → Pipeline shows the kanban board of all projects grouped by stage.

1Open the pipeline.

Five columns: Preparing, Open, Production, Done, Warranty. Each card shows project name, client, location, and payment progress bar.

2Search or scroll to find a project.

Type in the search bar to filter across all columns simultaneously.

3Click a card to open the project drawer.

The drawer shows client info, stage, and description. Click Open Project Page for the full detail view.

Task Approvals

When team members complete tasks that require management sign-off, they land in your approval queue. You'll also see a notification badge in the sidebar.

1Click the notification badge or go to Dashboard → Awaiting Approval.
2Expand the task card.

You'll see the task details, attached notes, document links, and who submitted it.

3Review, then update the status.

Set to Completed to approve, or add a note explaining why it needs revision and set it back to In Progress.

Some task approvals trigger the next task to unlock automatically. Completing them promptly keeps the project moving.

Projects View

Sidebar → Projects shows all projects in a searchable, filterable table.

Manager Notes

Each project row has a notes icon. Click it to open a text panel where you can leave private manager notes on the project — visible only to you and the superadmin.

Action Buttons

Each project row has quick-action buttons on the right:

  • F5 — Add or view quotation line items
  • F3 — Order materials for the project
  • View — Open the full project detail page

F5 — Adding Quotation Items

F5 is the form for recording what items were quoted and included in the project scope. This feeds into accountant reports.

1Find the project in the Projects list and click its F5 button.
2Click "+ Add Item" in the modal.
3Fill in the item details.

4Save. The line total calculates automatically.

Repeat for each item. The modal shows a running total at the bottom.

Project Detail Page

Click any project name to open its full detail page. It has two tabs.

Tasks & Items Tab

All project tasks in a board view. Expand any task card to update status, review attached documents, and read team notes.

Project Report Tab

A financial and operational summary visible to you and the superadmin:

  • Client info — name, phone, SED, quotation number, location.
  • Payments — contract value, collected, remaining, progress bar, full payment history.
  • Labour Hours — total regular and overtime hours from timesheets, with estimated cost.

Installation Team Assignment

When a project reaches the installation phase, it needs a team assigned before work can proceed. Unassigned projects are flagged in orange in the Projects list.

1Find the unassigned project (orange flag) in Projects view.
2Click "Assign Team".
3Select team members from the installation staff list.

You can select multiple members. Each selected member receives a notification about the assignment.

4Confirm. The team is now assigned and the installation tasks unlock.

Calendar

Sidebar → Calendar shows all project events. Use the tabs to filter by event type:

  • All — every event
  • Project Activity — fabrication and activity events
  • Payments — payment received and payment due (visible to you and superadmin only)
  • Installation — installation and delivery events
  • Material Delivery — delivery events

Click any event to view details. You can create new events by clicking a date on the calendar.

FAQ

How do I approve a task that needs my sign-off?

Go to your dashboard or click the notification badge in the sidebar. In the Awaiting Approval section, find the task and expand it. Review the details, then change the status to Completed to approve. Add a comment if you're sending it back for revision before changing the status to In Progress.

A project has no installation team — how do I assign one?

Look for the orange flag in the Projects list. Click the project row, then click Assign Team. Select the installation team members and confirm. The team gets notified and their installation tasks unlock.

How do I record a partial payment?

Go to Payments, expand the project, and click "+ Add Payment". Choose the appropriate type (e.g. Progressive or Delivery), enter the partial amount, set the status to Received, and submit. You can add multiple payments to the same project at any time.

How do I see how much has been collected vs what's outstanding?

Open the Project Detail page and click the Project Report tab. The Payments section shows the contract value, total collected, remaining balance, and a progress bar. You can also see the full payment history table below that.

How do I add a manager note on a project?

In the Projects view, find the project and click the notes icon on its row. A text panel opens where you can type and save your note. Notes here are private to managers and superadmin only.

How do I add a material order for a project?

In the Projects list, click the F3 button on the project row. Fill in the material details (name, quantity, unit, cost, supplier, required date) and submit. The order appears in the Materials report for the accountant.

What's the difference between F3 and F5?

F5 records what was quoted to the client — the line items and prices that make up the contract value. F3 is for ordering actual materials — it creates a purchase request for procurement and feeds the accountant's Payables report.
